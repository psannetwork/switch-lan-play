# switch-lan-play 初心者向けセットアップガイド

このガイドでは、`switch-lan-play` を動かすための手順を、全くの初心者の方でも分かるようにステップバイステップで解説します。

## はじめに：必要なもの
*   Windows PC
*   Nintendo Switch
*   インターネット環境

---

## ステップ1：パケットキャプチャ用ソフトウェアのインストール
`switch-lan-play` がSwitchの通信を中継するために、ネットワークの通信を読み取るソフトウェアが必要です。

かつては「WinPcap」が主流でしたが、現在はより新しく安全な **「Npcap」** を使用します。WinPcapは開発が終了しているため、必ずNpcapを入れてください。

1. [Npcap公式サイト](https://nmap.org/npcap/) にアクセスします。
2. 「Download」セクションから、最新のインストーラー（Npcap installer）をダウンロードします。
3. ダウンロードしたファイルを実行し、指示に従ってインストールしてください。
   *   **重要:** インストール途中のオプション設定で、**「Install Npcap in WinPcap API-compatible Mode」** という項目に必ずチェックを入れてください。（これがないと動かない場合があります）

---

## ステップ2：プログラムの入手
自分でビルド（構築）する必要がない場合、配布されている実行ファイル（バイナリ）を使用するのが最も簡単です。

1. GitHubの [releases ページ](https://github.com/spacemeowx2/switch-lan-play/releases) にアクセスします。
2. 最新のリリースから、Windows用の圧縮ファイル（`.zip` など）をダウンロードします。
3. ダウンロードしたファイルを解凍（展開）します。
4. 解凍したフォルダの中にある `lan-play-gui.exe` をダブルクリックして実行します。

---

## ステップ3：クライアントの設定と接続
1. GUIが起動したら、言語選択で「日本語」を選択してください。
2. 「クライアント」という項目に移動し、サーバーアドレスを入力します。
3. 「クライアント開始」ボタンを押せばPC側の準備は完了です。

---

## ステップ4：Switch側のネットワーク設定
PCと接続する前に、Switch本体の設定が必要です。以下の手順に従って設定してください。

**重要:** 詳細は [公式のインストールガイド](http://www.lan-play.com/install-switch) も合わせて確認してください。

### IPアドレス設定
IPアドレスは `10.13.0.1` から `10.13.255.254` の間で指定してください（ただし、`10.13.37.1` はゲートウェイとして使用するため設定しないでください）。また、友達と同じIPアドレスを使わないように注意してください。

*   **IPアドレス:** `10.13.XX.YY` (XXとYYは上記範囲で自由に設定)
*   **サブネットマスク:** `255.255.0.0`
*   **ゲートウェイ:** `10.13.37.1`

### DNS設定 (90DNSサーバー)
*   **プライマリDNS:** `163.172.141.219`
*   **セカンダリDNS:** `207.246.121.77`

---

## 注意事項：接続エラーについて
Switchのネットワーク設定完了後に接続テストを行うと、**接続エラーが表示されることがありますが、問題ありません。** そのまま設定を保存して、ゲームを起動してください。

---

## ステップ5：ゲーム内でのLANプレイ開始方法
Switchの設定が終わったら、実際にゲームを起動してLANプレイを開始します。ここでは「マリオカート8 デラックス」を例に説明します。

1. マリオカートを起動し、メインメニュー画面を表示します。
2. コントローラーの **「Rボタン + Lボタン + 左スティック押し込み」** を同時に押してください。
3. ローカル通信のメニュー表示が **「LANプレイ」** に切り替わります。
4. これを選択して、ルーム作成や参加を行えば、遠くの友達と遊ぶことができます。

※他のゲームでも、同様の操作でLANモードに切り替えられる場合があります。

バイナリが配布されていない場合や、自分で改造・ビルドしたい場合は、以下の手順で行います。MSYS2環境を使用します。

1. [MSYS2公式サイト](https://www.msys2.org/) からインストーラーをダウンロードし、インストールします。
2. インストール完了後、「MSYS2 UCRT64」または「MinGW 64-bit」ターミナルを起動します。
3. 以下のコマンドを実行して、必要なツールをインストールします。

```bash
pacman -Sy
pacman -S base-devel \
    mingw-w64-x86_64-toolchain \
    mingw-w64-x86_64-cmake \
    mingw-w64-x86_64-SDL2 \
    mingw-w64-x86_64-SDL2_image \
    git
```

4. `switch-lan-play` のフォルダに移動し、以下のコマンドを実行します。

```bash
bash build_win.sh
```

完了すると、`dist/` フォルダの中に `lan-play-gui.exe` が生成されます。
